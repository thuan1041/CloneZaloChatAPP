# Clone ChatApp Zalo

Clone ChatApp Zalo is a project aimed at replicating the functionality and user interface of Zalo, a popular messaging app, to enable seamless communication across web and mobile platforms.

**Note:** *These images are from previous updates. Since the last update, both the web and app versions have been enhanced with more features and functionalities.*
